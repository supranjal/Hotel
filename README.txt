# Jade Palace Hotel - Landing Page

Welcome to the official website of **Jade Palace Hotel**! This landing page is designed to provide a seamless and visually appealing experience for our visitors. Built with **HTML**, **CSS**, and primarily **Bootstrap**, the website ensures responsiveness and ease of navigation.

## Features

### 1. Navigation Bar
- A fully responsive and user-friendly navigation bar.
- Built using Bootstrap for a clean and modern design.
- Quick access to all sections of the website.

### 2. Facilities Section
- Highlights the top-notch amenities offered by Jade Palace Hotel.
- Includes details about the spa, gym, pool, and other luxurious facilities.

### 3. Room Section
- Showcases the variety of rooms available, from standard to deluxe suites.
- Includes images, descriptions, and key features of each room type.

## Technologies Used
- **HTML**: For structuring the content.
- **CSS**: For styling and enhancing the visual appeal.
- **Bootstrap**: For responsiveness and pre-designed components.
